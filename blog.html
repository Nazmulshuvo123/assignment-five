What is document object model (DOM)?

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree structure. It allows JavaScript to access and manipulate the structure, style, and content of the document. This enables dynamic web pages with interactive elements.

How do you select an element from the DOM?

To select an element from the DOM, use the getElementById() method. This method takes the element's ID as input and returns the element object.

What is event delegation in the context of the DOM, and why it is useful?

Event delegation in the DOM is a technique where a single event handler is attached to a parent element, and it listens for events that bubble up from its children. This is useful because it reduces the number of event handlers needed, improving performance and making code more maintainable.

What is event delegation in the context of the DOM?

Event delegation is a technique in JavaScript where you attach an event listener to a parent element instead of directly to each child element. This allows you to handle events for multiple child elements efficiently, especially when the number of child elements is dynamic or large.
